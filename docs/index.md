# neo-says

A snarky CLI fortune teller for developers.

## Installation

```bash
pip install neo-says
```

## Quick Start

```bash
neo-says          # Random developer quote
neo-says -t hacker  # Hacker theme
neo-says -l ko    # Korean quotes
neo-says tui      # Interactive browser
```

## Features

- 100+ developer quotes with multiple themes
- i18n support (English, Korean, Japanese)
- Interactive TUI browser
- Plugin/pack system
- REST API server
- Shell integration (bash/zsh/fish)
